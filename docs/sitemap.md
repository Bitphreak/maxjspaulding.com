# Site Map — Personal / Portfolio / Blog Sites

## Primary site: maxjspaulding.com
Purpose: Professional CV / Portfolio / Contact / “Business card + credibility” portal.

- **/ (Home / Landing Page)**
    - Snapshot summary: who you are (Entrepreneur, CTO/Architect, Founder of App Empire), tagline, maybe a headshot.
    - Highlights: short bullets or cards linking to sections below (Experience, Projects, Blog, Contact).
    - Optional social / external links: GitHub (bitphreak), LinkedIn, maybe email / contact.

- **/about**
    - Extended bio: background, philosophy (optional “guiding principles / values”), long-form blurb about your journey and what you do.
    - Possibly a timeline or “career history” — summarizing major milestones, roles, founding of App Empire, etc.
    - (Optional) downloadable résumé / CV (PDF link).

- **/experience** (or /work / /career)
    - Detailed history: past roles, responsibilities, tenure, achievements.
    - For your work at App Empire — short summary + link to App Empire site or relevant projects.
    - (Optional) Visuals / logos / key metrics or highlights.

- **/projects** (Portfolio)
    - Showcase of selected work: internal, public, open-source, consulting, side-projects.
    - Each project entry: name, short description, tech stack, outcome / impact, links (GitHub, live site, case study, etc.)
    - Possibly categorize (e.g. “Open Source / Personal”, “AI/Agent Systems”, “Consulting / Client Work”, “Infrastructure / Ops Tools”).

- **/contact**
    - Contact info (email, optional contact form or mailto link)
    - Links to GitHub, LinkedIn, other relevant profiles
    - (Optional) Short invitation to collaborate / hire / discuss — framing you as available for freelance / consulting

- **/blog** (optional, or redirect to bitphreak.dev)
    - If you want to reserve blog entries here (e.g. short announcements), or redirect to bitphreak.dev

- **/resume.pdf** (or similar)
    - A clean, downloadable résumé / CV document, for easy sharing
